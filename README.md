# wave-morphism

A shadcn-style design system for **wave-morphism** — UI that propagates, decays, and breathes like water.

Distilled from the objet d'art instrument pages (sea, tide, storm, pulse, pretext, jewel, watch, and more).

## Run

```bash
npm install
npm run dev
```

## Deploy

```bash
npx railway up
```

## Stack

Next.js 14 · TypeScript · Tailwind · Tidewater vellum tokens · Cormorant + JetBrains Mono
